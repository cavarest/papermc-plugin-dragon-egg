# Dragon Egg Lightning Plugin v0.1.0

## Initial Release

### Features
- Dragon Egg Lightning Ability with `/ability 1` command
- 3 sequential purple lightning strikes with 0.5-second intervals
- 1.5 hearts damage per strike (total 4.5 hearts if all connect)
- 60-second cooldown system
- HUD display with cooldown timer
- Target detection using ray tracing
- Comprehensive edge case handling

### Installation
1. Download the plugin JAR file below
2. Place it in your Paper server's `plugins/` directory
3. Restart your server
4. Players can use `/ability 1` with a Dragon Egg in offhand

### Requirements
- Paper 1.21.8 or higher
- Java 21 or higher
- Minecraft 1.21.8 or higher
